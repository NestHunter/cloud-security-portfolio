## About This Portfolio

I'm a cybersecurity professional transitioning into cloud architecture and AWS. My background is in security investigations and operations — I understand how systems get compromised, and now I'm learning how to build the infrastructure underneath them the right way.

This portfolio tracks everything I'm building and learning as I pursue the **AWS Solutions Architect Associate (SAA-C03)** certification and lay the groundwork for a career in cloud security engineering. It's not a finished showcase — it's a living record of a deliberate transition, built in public.

**CyberNest** is my long-term project in this space: a platform for cloud and security work that I'm planning to migrate onto AWS infrastructure. You'll see that thread develop here as the foundation gets built.
